String values for GuitarTuna

Before I was gifted a proper guitar tuner, I used the app guitartuna for tuning. For anyone using this app, below are tunings for some songs. The numbers you need on each string is noted inside the parentheses ().

If you want to obtain these values for your own song, here is the conversion:

  • Full-step = +20 (up) or -20 (down).
  • Half-step = +10 (up) or -10 (down).
